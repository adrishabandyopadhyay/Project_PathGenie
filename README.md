# PathGenie 🧠

An AI-based career recommendation system that predicts suitable career paths by combining resume classification and personality analysis.

## 🚀 Features
- Career prediction based on user inputs and resume data
- Personality-based recommendations (MBTI-inspired)
- NLP pipeline for text preprocessing and feature extraction
- Machine learning model for classification
- Personalized career suggestions with explanations

## 🛠️ Tech Stack
- Python
- Machine Learning
- NLP (TF-IDF, preprocessing)
- Logistic Regression
- Flask

## 📊 How It Works
1. User inputs resume / details  
2. Text is preprocessed and converted into features  
3. ML model predicts career domain  
4. System combines personality insights  
5. Outputs personalized career suggestions  

## 💡 Highlights
- Built ML pipeline from scratch  
- Integrated multiple models for hybrid recommendation  
- Focused on explainable outputs
